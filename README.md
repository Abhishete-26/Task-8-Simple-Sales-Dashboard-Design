# 📊 Sales Dashboard Project

## 📌 Objective
To build an interactive Power BI dashboard analyzing sales performance by Region, Category, and Month using the Superstore dataset.

---

## 🛠 Tools Used
- Power BI Desktop
- Superstore-Sales.csv Dataset

---

## 📈 Dashboard Features
- 📅 Monthly Sales Trend (Line Chart)
- 🌍 Sales by Region (Bar Chart)
- 🥧 Sales Distribution by Category (Donut Chart)
- 🎛 Interactive Region Slicer

---

## 🔍 Key Insights
1. West region generated the highest overall sales.
2. Sales peaked in November–December 2017, showing strong Q4 performance.
3. Technology category contributes the highest revenue share (36.4%).
4. Sales show noticeable monthly fluctuations, indicating seasonality.
